# Task Manager

This repository contains a single React task manager component in `TaskManager.jsx`.

## Features

- Add tasks with a title, optional description, priority, and due date
- Mark tasks as completed
- Filter tasks by status and priority
- Delete tasks with a short removal animation
- Save tasks in `localStorage`
- Highlight overdue tasks
- Support light and dark color schemes

## Project structure

- `TaskManager.jsx` – main React component and styling
- `README.md` – project documentation

## Notes

- Task data is stored in the browser under the `task-manager-items-v1` key
- The component is self-contained and includes its own styles
